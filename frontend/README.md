# DocMind Frontend

A production-quality UI for a document intelligence system.

## Features
- PDF Viewer with highlighting
- Document Hierarchy Tree
- Extracted Tables and Images Viewer
- JSON Data Inspector
- Summary and Keywords
- Mock Data Integration

## Tech Stack
- React (Vite)
- TailwindCSS
- Zustand (State Management)
- React PDF
- Lucide React (Icons)

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`

## Project Structure
- `src/components`: UI components (Sidebar, Toolbar, Viewers)
- `src/data`: Mock data files
- `src/store`: Zustand store
- `src/utils`: Utility functions (Axios)
